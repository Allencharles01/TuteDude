# Assignment 08 - Backend

## Quick start
1. `cd Assignment 08/backend`
2. `cp .env.example .env` and edit `.env` with `MONGO_URI` and `PORT`
3. `npm install`
4. `npm run dev` or `npm start`

## API base
`http://localhost:5000/api/todos`

## Endpoints
- GET `/api/todos`
- POST `/api/todos`
- GET `/api/todos/:id`
- PUT `/api/todos/:id`
- PATCH `/api/todos/:id/toggle`
- DELETE `/api/todos/:id`
